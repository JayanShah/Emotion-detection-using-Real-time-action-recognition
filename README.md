# Real-Time Pose Estimation (Skeleton Tracking) Demo

This Python script uses OpenCV to capture a webcam feed and Google's MediaPipe library to perform real-time pose estimation. It detects 33 body landmarks and overlays a 2D skeleton on the video.

This is a foundational component for projects involving action recognition, fitness tracking, or emotion detection from body language.

## Setup and Installation:
- create a virtual environment venv(version: python3.10.11)
- pip install -r requirements.txt


## Test
- run pose_tracker.py
